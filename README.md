## Stream Feeds Android tutorial

This is the final version of the tutorial. To follow along:

- Checkout and go to the initial commit:

```
# or download zip from https://github.com/GetStream/stream-feeds-android-tutorial/releases/tag/tutorial-start
git clone git@github.com:GetStream/stream-feeds-android-tutorial.git
git checkout tutorial-start
```

- Follow instructions from: https://getstream.io/activity-feeds/sdk/android/

If you just want to run the final tutorial, provide your credentials in `Credentials.kt`.
